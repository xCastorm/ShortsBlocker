# YouTube Shorts Remover & Redirect

A simple Chrome extension that **removes all YouTube Shorts elements and automatically redirects any Shorts URL** back to the YouTube homepage.

## Features
- Complete Shorts Removal

- The extension removes every Shorts-related UI element on YouTube, including:

- Shorts sidebar button

- Shorts shelves on the homepage and under videos

- Shorts thumbnails and tiles in recommendations and feeds

- Shorts items in search results

- Shorts player components

- Any <a> links pointing to /shorts/<id>

- Automatic Redirect

- Any visit to a Shorts URL (e.g., youtube.com/shorts/ID) is immediately redirected to the main YouTube homepage.


# Installation (Developer Mode)

1. **Download or clone** this repository:
   ```bash
   git clone https://github.com/xCastorm/ShortsBlocker.git
   
2. Open chrome://extensions/ or brave://extensions/ in your browser.

3. Enable Developer Mode (top right corner).

4. Click “Load unpacked”.

5. Select the folder where you cloned or unzipped the extension.
